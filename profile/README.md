## ℹ️ About

**Sites.Reviews** ([sites.reviews](https://sites.reviews)) is an independent catalog of companies and websites with user reviews and trust ratings — helping people decide who to trust online, and giving businesses a place to build reputation.

## 📦 Open-source
- **[sites-reviews-extension](https://github.com/SitesReviewsTrust/sites-reviews-extension)** — browser extension showing Sites.Reviews ratings as you browse. Install: [sites.reviews/extension](https://sites.reviews/extension)
