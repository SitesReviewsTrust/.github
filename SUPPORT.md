# Support

Thanks for using **[Sites.Reviews](https://sites.reviews)** — an independent
catalog of company and website reviews and trust ratings. This page explains
where to get help, depending on what you need.

> **Security issue?** Don't use the channels below — follow our
> [Security Policy](./SECURITY.md) and report it privately.

---

## 📖 Read the docs first

Most questions are answered in our documentation hub:

- **[sites-reviews-docs](https://github.com/SitesReviewsTrust/sites-reviews-docs)**
  — setup, the public API (`https://sites.reviews/api/public/v1`), the MCP
  server, the embeddable widget, and the browser extension.

## 🐞 Found a bug or want a feature?

Open an issue **in the repository it affects**, using the issue forms:

| You're using… | File issues in |
|---|---|
| The MCP server (AI assistant integration) | [sites-reviews-mcp](https://github.com/SitesReviewsTrust/sites-reviews-mcp/issues) |
| The public API or embeddable widget | [sites-reviews-api](https://github.com/SitesReviewsTrust/sites-reviews-api/issues) |
| The browser extension | [sites-reviews-extension](https://github.com/SitesReviewsTrust/sites-reviews-extension/issues) |
| The docs | [sites-reviews-docs](https://github.com/SitesReviewsTrust/sites-reviews-docs/issues) |

Please **search existing issues** before opening a new one, and fill in the
issue form completely — clear reproduction steps get fixes merged faster.

## 💬 Questions & discussion

For "how do I…?", integration help, ideas, and general conversation, use
**[GitHub Discussions](https://github.com/orgs/SitesReviewsTrust/discussions)**
for the organization. Discussions are better than issues for open-ended
questions.

## 🤖 Telegram

For quick questions, account/listing help, or to reach us directly:

- **[@SitesReviews_bot](https://t.me/SitesReviews_bot)**

This is also the place to **dispute or correct a specific listing or trust
rating** — that's a catalog/data matter, not a code issue, so please don't open
a repo issue for it.

## 🌐 Website & tools

- **Website** — [sites.reviews](https://sites.reviews)
- **Check any site** (trust score / scam check) — search a domain on
  [sites.reviews](https://sites.reviews)
- **Browser extension** — see a Trust Score on any site you visit:
  [sites.reviews/extension](https://sites.reviews/extension)

## ⏱️ What to expect

This is an open-source project maintained by a small team. We watch issues and
discussions and respond as quickly as we can, but there is **no guaranteed
response time** for general support. Well-described, reproducible reports get
attention first. Thank you for your patience.
